## Sobre o Projeto

Este projeto apresenta uma página web estruturada com listas estilizadas e um formulário de cadastro, demonstrando boas práticas de organização e estilização.

Foi desenvolvido com o objetivo de praticar:

- Estruturação semântica com HTML
- Estilização com CSS
- Organização de layout com Grid
- Criação de formulários

---

## Funcionalidades

- Listas não ordenadas, ordenadas e de definição
- Layout moderno com uso de CSS Grid
- Design responsivo básico
- Formulário com:
  - Nome
  - Email
  - Marca favorita
  - Interesses
  - Gênero
  - Mensagem
